# Project templates
Standardized project templates for Ikon Technologies product development.

- **lean-canva.md** — Discovery: validate whether the initiative is worth building.
- **prd-template.md** — Definition: full PRD including scope and stakeholder sign-off.
- **scope-agreement-stakeholders.md** — What PM offers internal stakeholders (e.g. Ops VP) to agree on project and in/out of scope.
